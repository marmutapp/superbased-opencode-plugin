---
description: Check and manage SuperBased authentication
---

Use `superbased_auth` with `action: "status"` to check authentication state.

Report: signed in or not, email, plan tier, license status.

If not authenticated:
- **Desktop app**: sign in via SuperBased app > Settings > Account
- **Headless/CLI**: run `superbased auth login` in terminal for browser-based sign-in
